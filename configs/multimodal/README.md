# Multimodal configs

Put LiDAR-camera fusion experiment configs here, for example:

- nuScenes multimodal detection configs
- `fusion_detection_template.py`: editable starter template for KITTI multimodal experiments
- `mvxnet_kitti_3class.py`: runnable KITTI LiDAR-camera fusion baseline aligned with upstream MVXNet
- multimodal attack evaluation configs
- multimodal defense evaluation configs
